## Basic Syntax 基本语法{#basic-syntax}

### 概述 Overview

几乎所有 Markdown 应用程序都支持原始 Markdown 设计文档中概述的基本语法。Markdown 处理器之间存在细微的差异和出入，这些差异和出入在可能的情况下都会以内联方式注明。
